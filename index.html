Geno-PanPipe Package for Genomic Variant Calling

Introduction

Geno-PanPipe is a software package designed to execute bioinformatics pipelines, with a particular focus on genome variant calling. Pipelines are implemented as modules that can be executed by means of the PanPipe Workflow Manager. Geno-PanPipe also provides some utilities useful to automate experiments on complete datasets.

Features

The package incorporates the following features:

  • Execution of pipelines processing normal-tumor files.
  • Automation of bam file downloading.
  • Pipeline steps automating bam file manipulation.
  • Steps implementing single nucleotide/copy number/structural variant calling.
  • Steps implementing microsatellite instability (MSI) analysis.
  • Parallel versions of multiple variant calling tools.

Distribution Details

Geno-PanPipe has been coded using shell-scripting. Geno-PanPipe is known to compile on Unix-like and Windows (using Cygwin) systems. See the "Documentation and Support" section of these instructions if you experience problems during compilation.

Installation

Basic Installation Procedure

To install Geno-PanPipe, first you need to install the autotools (autoconf, autoconf-archive, automake and libtool packages in Ubuntu). If you are planning to use Geno-PanPipe on a Windows platform, you also need to install the Cygwin environment. Alternatively, the tool can also be installed on Mac OS X systems using MacPorts.

Pipeline modules provided by Geno-PanPipe are designed to be executed by the PanPipe workflow manager. Therefore, PanPipe and all of its requirements should also be installed before installing Geno-PanPipe.

On the other hand, some of the functionality incorporated by Geno-PanPipe requires the previous installation of third-party software (see below).

Once the autotools are available (as well as other required software such as Cygwin or MacPorts), you can proceed with the installation of the tool by following the next sequence of steps:

  1. Obtain the package using git:

    $ git clone https://github.com/daormar/geno-panpipe.git

    Or download it in a zip file

  2. cd to the directory containing the package's source code and type ./reconf.

  3. Type ./configure --with-panpipe=<DIR> to configure the package (where <DIR> is the build directory of the PanPipe package).

  4. Type make to compile the package.

  5. Type make install to install the programs and any data files and documentation.

  6. You can remove the program binaries and object files from the source code directory by typing make clean.

By default the files are installed under the /usr/local/ directory (or similar, depending on the OS you use); however, since Step 5 requires root privileges, another directory can be specified during Step 3 by typing:

 $ configure --with-panpipe=<DIR> --prefix=<absolute-installation-path>

For example, if "user1" wants to install the Geno-PanPipe package in the directory /home/user1/geno-panpipe, the sequence of commands to execute should be the following:

 $ ./reconf
 $ configure --prefix=/home/user1/geno-panpipe
 $ make
 $ make install

The installation directory can be the same directory where the Geno-PanPipe package was decompressed.

Alternative Installation Options

The Geno-PanPipe configure script can be used to modify the toolkit behavior. Here is a list of current installation options:

  • --with-icgcscor=<DIR>: allows Geno-PanPipe to use the ICGC score client. The ICGC score client can be used to interact with cloud repositories.
  • --with-aspera=<DIR>: enables use of the Aspera commercial file transfer protocol to download bam files.
  • --with-egadecrypt=<DIR>: allows Geno-PanPipe to access decryption functionality provided by the EGACryptor client.

Third Party Software

ICGC Score Client

The ICGC score client allows to download content, including bam files from cloud repositories. The client can be downloaded after clicking in this link.

ASPERA Connect

ASPERA Connect is a file transfer protocol that may provide faster transfer speeds with respect to the standard FTP protocol. The ASPERA download client can be used by Geno-PanPipe to download bam files. The client can be downloaded here.

EGA Decryptor

EGACryptor is a JAVA based client providing encryption and decryption functionality. It is used by Geno-PanPipe to decrypt information downloaded from EGA repositories.
